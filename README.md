<h1>ToRemember<h1></h1>

<h3>Description<h3></h3>


Program that allows user to create memorylists and add notes to those memorylists to not miss important things in their lives. <br>
<h3>Usage<h3></h3>

Run the program in IntelliJ IDEA.<br>
1) Locate the server class (`SingleServerListener.java`) in the project explorer.<br>
2) Right-click the file and select `Run 'SingleServerListener.main()'`<br>
3) Locate the program class (`ToRemember.java`) in the project explorer.<br>
4) Right-click the file and select `Run 'ToRemember.main()'`<br>
<br>

**Example Usage:**<br>
This program is currently only for terminal use, follow the terminaloutputs to navigate through program. <br>
Create memorylist/note, remove memorylist/note, sort notes by priority, show all notes by category <br>


<h3>Design Patterns<h3></h3>
  
Program is implemented with __MVC__ pattern to make it easier with implementing other views, like SWING or WEB based. <br>
__Singleton__ is also used to make sure that user is communicating to the right server and the server is accessing the same MySQL database <br>
